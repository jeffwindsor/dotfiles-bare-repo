# Doom Emacs

## Useful Commands to Learn

### General

- SPC f p : open configuration files
- SPC f f : find file
- SPC o t : toggle terminal

- SPC ,   : switch to buffer
- SPC b k : kill buffer
- SPC b N : new buffer

- z z     : center cursor line in display


### Projects

- SPC SPC : find file in project
- SPC o p : open project sidebar
